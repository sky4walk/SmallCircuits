Tag System → Cyclic Tag (s-aware)

Konvertiert ein Tag-System mit Löschzahl s in ein Cyclic Tag System — korrekt für echtes p≥2. Der Trick (Cook §1.3): die zyklische Liste wird auf s·|Φ| Appendants erweitert, davon (s−1)·|Φ| leer. So werden pro s-tag-Schritt genau s Symbole gelöscht, aber nur die Produktion des ersten angehängt. Anders als der 1-tag-Encoder emuliert das auch p=2-Systeme korrekt (z.B. den Output der polynomialen TM→2-tag-Stufe).
