Mathematischer Hintergrund

✕ Multiplikation — C/D-Skala

Beide Skalen sind logarithmisch eingeteilt. Die Position von Wert x liegt bei:

x → log₁₀(x) × L

Verschiebt man die C-Skala um log₁₀(a), addiert man Logarithmen:

log(a) + log(b) = log(a·b)

Der Cursor zeigt auf der D-Skala direkt das Produkt ab.

÷ Division — CI/D-Skala

Die CI-Skala (C-inverted) läuft rückwärts — von rechts nach links. Damit entspricht die Verschiebung einem Subtrahieren des Logarithmus:

log(a) - log(b) = log(a/b)

Slide auf b der CI-Skala setzen, Cursor auf a der D-Skala — Ergebnis a/b erscheint auf D.

+ Addition — L-Skala

Die L-Skala ist linear (0 bis 1). Hier findet keine logarithmische Transformation statt:

Position(x) = x × L

Slide um Wert a verschieben, Cursor auf Wert b — die rote Linie zeigt direkt a + b auf der Skala.

a + b = (a+b) × L / L

− Subtraktion — LI-Skala

Die LI-Skala (L-inverted) ist eine gespiegelte lineare Skala — analog zur CI-Skala, aber ohne Logarithmus:

Position(x) = (1 − x) × L

Slide auf b der LI-Skala setzen — damit wird b von a subtrahiert:

a − b = a + (−b)

Der Cursor zeigt auf der L-Skala das Ergebnis a − b.

Warum funktioniert das? Der Rechenschieber macht aus Multiplikation eine Addition durch die Eigenschaft des Logarithmus: log(a·b) = log(a) + log(b). Den Logarithmus physisch als Länge darzustellen und dann Längen zusammenzufügen — das ist das geniale Prinzip, das William Oughtred 1622 entwickelte. Vor dem Taschenrechner waren Rechenschieber das wichtigste Werkzeug von Ingenieuren, Wissenschaftlern und Piloten. Ein geübter Benutzer konnte damit in Sekunden auf 3 Stellen genau rechnen.
