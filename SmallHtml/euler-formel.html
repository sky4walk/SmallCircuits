Die Euler-Formel

eix = cos(x) + i·sin(x) — live auf dem Schirm. Regler ziehen, Formel begreifen.

01 — Worum geht es

Zwei bekannte Dinge, eine Formel

Die Taylor-Reihen von Sinus und Kosinus kennst du bereits, ebenso die Zahl e. Leonhard Euler (1707–1783) verband beides über die komplexen Zahlen zu einer einzigen Gleichung. Der Rest dieser Seite ist ein Oszilloskop dafür: Der Zeiger auf dem Schirm unten ist eix, für jeden Winkel x, live berechnet — keine Animation, echte Mathematik.

02 — Komplexe Zahlen

Die erfundene Zahl i

√(-1) existiert nicht unter den reellen Zahlen — also erfindet man i mit der Eigenschaft i·i = -1. Eine komplexe Zahl z = a + b·i hat dann einen reellen Anteil a und einen imaginären Anteil b. Multiplikation mit i dreht einen Zeiger in der Ebene um genau 90° — das ist der Grund, warum eix überhaupt auf einem Kreis läuft.

04 — Taylor-Reihe von e^x

Die einfachste Taylor-Reihe überhaupt

Jede Ableitung von ex ist wieder ex, und e0 = 1. Also sind alle Koeffizienten 1:

e^x = 1 + x + x²/2! + x³/3! + x⁴/4! + x⁵/5! + …

05 / 06 — ix einsetzen, sortieren

Der entscheidende Schritt

x wird durch ix ersetzt. Die Potenzen von i wiederholen sich alle vier Schritte: i, -1, -i, 1, i, …

e^(ix) = 1 + ix − x²/2! − ix³/3! + x⁴/4! + ix⁵/5! − …

Reelle und imaginäre Terme getrennt ergeben genau die Taylor-Reihen von Kosinus und Sinus:

e^(ix) = ( 1 − x²/2! + x⁴/4! − … ) + i · ( x − x³/3! + x⁵/5! − … ) = cos(x) + i·sin(x)

Genau diese Partialsummen zeichnet der Schirm oben als Vektorkette, wenn du „Taylor-Approximation“ aktivierst.

07 — Die schönste Formel der Mathematik

Der Sonderfall x = π

Bei x = π gilt cos(π) = -1 und sin(π) = 0. Damit wird aus der Euler-Formel die Euler-Identität — fünf fundamentale Konstanten in einer Gleichung:

e^(iπ) + 1 = 0

„Die schönste Gleichung der Mathematik“ — Richard Feynman

09 — Zusammenfassung

Der Weg zur Euler-Formel

  1. Taylor-Reihe von e^x aufschreiben
    e^x = 1 + x + x²/2! + x³/3! + …
  2. x durch ix ersetzen
    Potenzen von i berechnen: sie wechseln zwischen i, -1, -i, 1, …
  3. Terme mit und ohne i sortieren
    Es entstehen zwei separate Reihen
  4. Taylor-Reihen von cos und sin erkennen
    Reelle Terme → cos(x), imaginäre Terme → sin(x)
  5. Ergebnis
    e^(ix) = cos(x) + i·sin(x) — und für x = π folgt die Euler-Identität
