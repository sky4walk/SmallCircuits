Kapitel 1

Säure-Base-Reaktionen

Säuren und Basen begegnen uns überall: im Essig, in Reinigungsmitteln, in der Magensäure oder im Backpulver. In diesem Kapitel lernst du, wie sie miteinander reagieren und wie man ihre Konzentration genau bestimmt.

1.1 Neutralisationsreaktionen

Wenn eine Säure und eine Base zusammentreffen, neutralisieren sie sich gegenseitig – die sauren und basischen Eigenschaften heben sich auf. Das Produkt ist immer Wasser und ein Salz.

Säure + Base → Wasser + Salz Allgemeine Gleichung der Neutralisation
HCl + NaOH → H₂O + NaCl Salzsäure + Natronlauge → Wasser + Kochsalz

Was passiert auf Teilchenebene?

Um die Reaktion wirklich zu verstehen, schauen wir auf die einzelnen Ionen. Säuren und Basen zerfallen in Wasser in ihre Ionen – die eigentliche Neutralisation ist dann immer dieselbe Kernreaktion:

Ionengleichungen

Salzsäure dissoziiert:

HCl + H₂O ⇌ H₃O⁺ + Cl⁻

Natronlauge dissoziiert:

NaOH → Na⁺ + OH⁻

Kernreaktion (die eigentliche Neutralisation):

H₃O⁺ + OH⁻ → 2 H₂O

Na⁺ und Cl⁻ sind Zuschauer-Ionen – sie nehmen nicht direkt an der Reaktion teil und bleiben in Lösung.

Bei jeder Neutralisation reagieren Oxoniumionen (H₃O⁺) mit Hydroxidionen (OH⁻) zu Wasser. Das ist immer dieselbe Kernreaktion – egal welche Säure und welche Base. Wichtig: Säure-Base-Reaktionen sind umkehrbar (⇌ = Gleichgewichtspfeil)!

Weitere Beispiele

Neutralisationsgleichungen im Überblick
HBr + NaOH → H₂O + NaBrBromwasserstoff + Natronlauge
H₂SO₄ + 2 KOH → 2 H₂O + K₂SO₄Schwefelsäure + Kalilauge – Achtung: 2 KOH nötig, da H₂SO₄ zweiprotonig ist!
CH₃COOH + NaOH → NaCH₃COO + H₂OEssigsäure + Natronlauge
3 HCl + Al(OH)₃ → 3 H₂O + AlCl₃Salzsäure + Aluminiumhydroxid – Achtung: 3 HCl nötig, da Al(OH)₃ dreibasig ist!

Sodbrennen: Bei zu viel Magensäure (HCl) helfen Antazida mit Al(OH)₃ oder Mg(OH)₂ – sie neutralisieren die Säure: 3 HCl + Al(OH)₃ → 3 H₂O + AlCl₃.

1.2 Ampholyte

Manche Stoffe können je nach Reaktionspartner sowohl als Säure als auch als Base reagieren. Man nennt sie Ampholyte.

Voraussetzung für einen Ampholyten: mindestens ein positiv polarisiertes H-Atom (kann abgegeben werden) UND ein freies Elektronenpaar (kann ein Proton aufnehmen).

Das beste Beispiel: Wasser (H₂O)

Wasser ist der bekannteste Ampholyt – es reagiert unterschiedlich je nachdem, mit welchem Stoff es in Kontakt kommt:

Wasser als BASE

Reaktion mit HCl:

HCl + H₂O → H₃O⁺ + Cl⁻

Wasser nimmt ein Proton (H⁺) vom HCl auf → Protonenakzeptor = Base

Wasser als SÄURE

Reaktion mit NH₃:

NH₃ + H₂O → NH₄⁺ + OH⁻

Wasser gibt ein Proton (H⁺) an NH₃ ab → Protonendonator = Säure

Weitere Ampholyte

Weitere Beispiele für Ampholyte

Hydrogencarbonat (HCO₃⁻):

Als Säure: HCO₃⁻ + H₂O ⇌ CO₂ + H₃O⁺
Als Base: HCO₃⁻ + H₂O ⇌ H₂CO₃ + OH⁻

Aminosäuren sind ebenfalls Ampholyte – das erklärt viele biologische Prozesse im Körper.

1.3 Titration

Die Titration ist ein Messverfahren, um die unbekannte Konzentration einer Säure oder Base zu bestimmen. Man gibt eine Lösung mit bekannter Konzentration (= Maßlösung) tropfenweise zu, bis der Äquivalenzpunkt erreicht ist.

Wichtige Fachbegriffe

Maßlösung: Lösung mit bekannter Konzentration (z.B. NaOH, c = 0,1 mol/L)

Vorlage: Lösung mit unbekannter Konzentration (z.B. HCl)

Indikator: Farbstoff, der je nach pH-Wert die Farbe wechselt (z.B. Bromthymolblau: orange → grün)

Äquivalenzpunkt: Punkt, an dem n(H₃O⁺) = n(OH⁻) – gleiche Stoffmengen haben reagiert

Bürette: Messgerät zum genauen Ablesen des verbrauchten Volumens der Maßlösung

Schlüsselformel: n = c · V (Stoffmenge = Konzentration × Volumen)

Rechenweg Schritt für Schritt

Beispiel aus dem Unterricht: 20 mL NaOH (c = 0,1 mol/L) neutralisieren 20 mL HCl. Gesucht: c(HCl)?

Rechenweg Titration
①
Gegebene Werte:
c(NaOH) = 0,1 mol/L  |  V(NaOH) = 20 mL = 0,020 L  |  V(HCl) = 20 mL = 0,020 L
②
Reaktionsgleichung:
NaOH(aq) + HCl(aq) → H₂O(l) + NaCl(aq)
③
Stoffmengenverhältnis:
n(NaOH) : n(HCl) = 1 : 1  →  n(NaOH) = n(HCl)
④
Formel n = c · V einsetzen:
c(NaOH) · V(NaOH) = c(HCl) · V(HCl)
⑤
Berechnen:
0,1 · 0,020 = c(HCl) · 0,020
c(HCl) = 0,1 mol/L ✓

Weiteres Beispiel (Kohlensäure): 25 mL KOH (c = 0,05 mol/L) titrieren 30 mL H₂CO₃. Da die Gleichung H₂CO₃ + 2 KOH → K₂CO₃ + 2 H₂O ein 1:2-Verhältnis hat, gilt: n(KOH) = 2 · n(H₂CO₃). Daraus folgt c(H₂CO₃) = (0,05 · 0,025) / (2 · 0,030) ≈ 0,021 mol/L.

Kapitel 2

Redoxreaktionen

Bei Redoxreaktionen wandern keine Protonen, sondern Elektronen. Diese Art von Reaktion erklärt Rosten, das Verbrennen, das Laden von Akkus und vieles mehr.

2.1 Grundlagen: Oxidation und Reduktion

Oxidation

= Elektronenabgabe

Ein Teilchen gibt Elektronen ab.
Oxidationszahl steigt.

Mg → Mg²⁺ + 2e⁻

Mg wird oxidiert (OZ: 0 → +II)

Reduktion

= Elektronenaufnahme

Ein Teilchen nimmt Elektronen auf.
Oxidationszahl sinkt.

O₂ + 4e⁻ → 2 O²⁻

O₂ wird reduziert (OZ: 0 → -II)

Oxidation und Reduktion laufen IMMER gleichzeitig ab – was ein Teilnehmer an Elektronen abgibt, nimmt der andere sofort auf. Man kann sie nicht trennen. Deshalb: Redox = Reduktion + Oxidation.

Klassisches Beispiel: Knallgasreaktion

2 H₂ + O₂ → 2 H₂O Stark exotherm – lauter Knall, Stichflamme
Teilreaktionen der Knallgasreaktion

Oxidation (Wasserstoff gibt Elektronen ab):

2 H₂ → 4 H⁺ + 4e⁻

Reduktion (Sauerstoff nimmt Elektronen auf):

O₂ + 4e⁻ → 2 O²⁻

Gesamtreaktion: 2 H₂ + O₂ → 2 H₂O

Verbrennung von Magnesium

Das zweite Demonstration-Experiment: Magnesiumband verbrennt mit intensivem Weißlicht zu Magnesiumoxid.

2 Mg + O₂ → 2 MgO Stark exotherm – intensive weiße Leuchterscheinung
Teilreaktionen
Ox: 2 Mg → 2 Mg²⁺ + 4e⁻
Red: O₂ + 4e⁻ → 2 O²⁻

2.2 Oxidationszahlen

Oxidationszahlen (OZ) sind eine Art Buchhaltung für Elektronen. Sie zeigen, wie viele Elektronen ein Atom in einer Verbindung im Vergleich zum freien Element aufgenommen oder abgegeben hat. Sie werden mit römischen Zahlen über dem Elementsymbol angegeben.

Kernregel: Wenn sich Oxidationszahlen ändern → Redoxreaktion! Wenn sich keine OZ ändert → andere Reaktion (z.B. Säure-Base).
Regeln zur Bestimmung von Oxidationszahlen
1
Atome in reinen Elementen: OZ = 0   (O₂, Fe, Cl₂, H₂, ...)
2
Einfache Ionen: OZ = Ionenladung   (Na⁺ → +I, Cl⁻ → -I, Ca²⁺ → +II)
3
Summe aller OZ in einem Molekül = 0; in einem Molekül-Ion = Ladung des Ions
4
Sauerstoff hat meist OZ -II (Ausnahme: Peroxide wie H₂O₂ → OZ -I)
5
Wasserstoff hat meist OZ +I (Ausnahme: Metallhydride wie NaH → OZ -I)
6
Fluor hat immer OZ -I (elektronegativstes Element)
7
Metalle haben in Verbindungen immer positive OZ
8
Chlor, Brom, Iod haben in Verbindungen meist OZ -I

Beispielrechnung: OZ von Schwefel in H₂SO₄

OZ-Berechnung für H₂SO₄
①
OZ(H) = +I (Regel 5), OZ(O) = -II (Regel 4), Summe = 0 (Regel 3)
②
2·(+1) + x + 4·(-2) = 0
③
2 + x - 8 = 0  →  x = +6 → OZ(S) = +VI ✓

2.3 Redoxgleichungen aufstellen

Das systematische Aufstellen von Redoxteilgleichungen ist eine wichtige Prüfungskompetenz. Man folgt dabei immer demselben Ablauf:

Schritt-für-Schritt-Anleitung
①
Ausgangsstoff und Endstoff hinschreiben
②
Oxidationszahlen bestimmen und markieren (z.B. mit kleinen Zahlen über den Elementen)
③
Elektronen in Teilgleichungen zuordnen – wie viele Elektronen wechseln pro Atom?
④
Ladungen ausgleichen: mit H₃O⁺ (saures Medium) oder OH⁻ (alkalisches Medium)
⑤
Teilchenzahl durch H₂O ausgleichen
⑥
Teilgleichungen erweitern, sodass gleiche Elektronenanzahl in beiden Gleichungen steht
⑦
Beide Teilgleichungen addieren, Elektronen kürzen, ggf. H₂O kürzen

Beispiel: Unedles Metall mit Säure (Mg + HCl)

Mg + 2 HCl → MgCl₂ + H₂ Gasentwicklung (H₂), exotherm – Mg löst sich auf
Nachweis der Redoxreaktion
Ox: Mg → Mg²⁺ + 2e⁻     (OZ: 0 → +II, Elektronenabgabe)
Red: 2 H₃O⁺ + 2e⁻ → H₂ + 2 H₂O    (OZ: +I → 0, Elektronenaufnahme)
Redox: Mg + 2 H₃O⁺ → Mg²⁺ + H₂ + 2 H₂O
Wichtiger Unterschied: Wenn Salze (z.B. CaCO₃) mit Säuren reagieren und Gasentwicklung auftritt, ändern sich die Oxidationszahlen NICHT → das ist eine Säure-Base-Reaktion, keine Redoxreaktion!

Beispiel: Oxidation von Ethanol

Alkohole können durch Oxidationsmittel in Aldehyde umgewandelt werden. Das wurde mit ammoniakalischer Kupfersulfatlösung gezeigt:

Ethanol → Ethanal (Kupfer-II-Test)

Beobachtung: Blaue Cu²⁺-Lösung wird rot (Cu₂O, Kupfer(I)-oxid)

Ox: CH₃CH₂OH → CH₃CHO + 2H⁺ + 2e⁻Ethanol wird zu Ethanal oxidiert
Red: 2 Cu²⁺ + 2e⁻ + 2 OH⁻ → Cu₂O + H₂OKupfer(II) wird zu Kupfer(I) reduziert

2.4 Redox in Abhängigkeit vom pH-Wert

Das Ergebnis einer Redoxreaktion kann sich je nach Säure- oder Basezusatz ändern. Das wurde mit Kaliumpermanganat (KMnO₄) – einer violetten Lösung – gezeigt:

Saures Medium (pH < 7)

KMnO₄ (violett) → Mn²⁺ (farblos)

Die violette Farbe verschwindet.

Red: MnO₄⁻ + 8H₃O⁺ + 5e⁻ → Mn²⁺ + 12H₂O

Ausgleich mit H₃O⁺

Alkalisches Medium (pH > 7)

KMnO₄ (violett) → MnO₂ (braun)

Brauner Niederschlag entsteht.

Red: MnO₄⁻ + 3e⁻ + 2H₂O → MnO₂ + 4OH⁻

Ausgleich mit OH⁻

Vollständiges Beispiel (saures Medium): MnO₄⁻ + SO₃²⁻

Redoxgleichung Permanganat / Sulfit (saures Medium)
Red.
MnO₄⁻ + 8H₃O⁺ + 5e⁻ → Mn²⁺ + 12H₂O
(Mn: +VII → +II, nimmt 5e⁻ auf)
Ox.
SO₃²⁻ + 3H₂O → SO₄²⁻ + 2e⁻ + 2H₃O⁺
(S: +IV → +VI, gibt 2e⁻ ab)
×2 / ×5
Reduzierte gleich viele Elektronen: Red. ×2, Ox. ×5 → jeweils 10e⁻
Ges.
2 MnO₄⁻ + 5 SO₃²⁻ + 6H₃O⁺ → 2 Mn²⁺ + 5 SO₄²⁻ + 9 H₂O

2.5 Vergleich: Säure-Base vs. Redox

Merkmal Säure-Base Redox
Was wandert? Protonen (H⁺) Elektronen (e⁻)
Wer gibt ab? Protonendonator (Säure) Reduktionsmittel (wird oxidiert)
Wer nimmt auf? Protonenakzeptor (Base) Oxidationsmittel (wird reduziert)
Ändern sich OZ? Nein Ja – immer!
Produkte Wasser + Salz Oxid / Salz / Gas – je nach Reaktion
Beispiel HCl + NaOH → NaCl + H₂O Mg + 2HCl → MgCl₂ + H₂
Trick für die Klausur: Zuerst Oxidationszahlen bestimmen. Ändern sie sich? → Redox. Ändern sie sich nicht, aber es wandern Protonen? → Säure-Base. Beides möglich? → Nein, es ist immer genau eines von beiden!
