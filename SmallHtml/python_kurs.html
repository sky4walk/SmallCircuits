Einführung in PythonWas ist Python und warum lohnt es sich?

Python ist eine der beliebtesten Programmiersprachen der Welt. Sie wird eingesetzt für Webseiten (Instagram, YouTube), Datenanalyse, künstliche Intelligenz, Automatisierung und vieles mehr.

💡 Warum Python?

Python-Code liest sich fast wie normales Englisch. Als Anfänger lernst du dadurch schneller die Konzepte des Programmierens, ohne dich mit komplizierter Syntax zu kämpfen.

Dein erstes Programm

Jeder Programmierer beginnt mit dem berühmten "Hello, World!" – einem Programm, das einfach nur Text auf dem Bildschirm ausgibt.

python hello.py

print() ist eine eingebaute Funktion in Python. Sie gibt alles, was du in die Klammern schreibst, auf dem Bildschirm aus. Text muss immer in Anführungszeichen stehen (einfach ' oder doppelt ").

Mit if kannst du deinem Programm beibringen, je nach Situation anders zu reagieren. Einrückung ist in Python entscheidend! Alles, was zu einem if-Block gehört, muss 4 Leerzeichen eingerückt sein.

Dateien lesen & schreibenDaten dauerhaft speichern

Mit Python kannst du ganz einfach Dateien lesen und schreiben. Das with-Statement sorgt dafür, dass die Datei automatisch geschlossen wird.

pythondateien.py
# Datei schreiben (überschreibt vorhandene)
with open("notizen.txt", "w", encoding="utf-8") as datei:
    datei.write("Erste Zeile\n")
    datei.write("Zweite Zeile\n")

# Datei anhängen (append)
with open("notizen.txt", "a", encoding="utf-8") as datei:
    datei.write("Dritte Zeile\n")

# Datei lesen
with open("notizen.txt", "r", encoding="utf-8") as datei:
    inhalt = datei.read()
    print(inhalt)

# Zeile für Zeile lesen
with open("notizen.txt", "r", encoding="utf-8") as datei:
    for zeile in datei:
        print(zeile.strip())  # .strip() entfernt \n
ModusBedeutung
"r"Lesen (Datei muss existieren)
"w"Schreiben (überschreibt oder erstellt)
"a"Anhängen (fügt am Ende hinzu)
"x"Neu erstellen (Fehler wenn schon vorhanden)

def sichere_division():
    try:
        a = int(input("Zähler: "))
        b = int(input("Nenner: "))
        print(f"Ergebnis: {a / b}")

    except ValueError:
        print("Fehler: Bitte nur Zahlen eingeben!")

    except ZeroDivisionError:

    except Exception as e:

    else:
        print("Kein Fehler aufgetreten ✅")

    finally:
        print("Berechnung abgeschlossen.")  # läuft IMMER

Klassen & Objekte (OOP)Eigene Datentypen erstellen

Objektorientierte Programmierung (OOP) ermöglicht es, die echte Welt in Code abzubilden. Eine Klasse ist wie ein Bauplan; ein Objekt ist das, was nach dem Bauplan gebaut wurde.

pythonoop.py
class Hund:
    """Repräsentiert einen Hund."""

    # __init__ wird aufgerufen wenn ein neues Objekt erstellt wird
    def __init__(self, name, rasse, alter):
        self.name  = name    # Attribute des Objekts
        self.rasse = rasse
        self.alter = alter

    # Methode (Funktion eines Objekts)
    def bellen(self):
        print(f"{self.name} bellt: Wuff! Wuff! 🐕")

    def vorstellen(self):
        print(f"Ich heiße {self.name}, bin ein {self.rasse} und {self.alter} Jahre alt.")

    # __str__ definiert die Textdarstellung
    def __str__(self):
        return f"Hund({self.name})"


# Objekte erstellen (Instanziierung)
bello = Hund("Bello", "Labrador", 3)
lassie = Hund("Lassie", "Collie", 5)

bello.bellen()
lassie.vorstellen()
print(bello)     # Hund(Bello)
print(bello.alter)  # 3
Bello bellt: Wuff! Wuff! 🐕
Ich heiße Lassie, bin ein Collie und 5 Jahre alt.
Hund(Bello)

Vererbung

Eine Klasse kann von einer anderen erben und ihre Eigenschaften übernehmen:

python
class Tier:
    def __init__(self, name):
        self.name = name

    def essen(self):
        print(f"{self.name} isst.")


class Katze(Tier):   # erbt von Tier
    def __init__(self, name, farbe):
        super().__init__(name)  # Eltern-__init__ aufrufen
        self.farbe = farbe

    def miauen(self):
        print(f"{self.name} sagt: Miau! 🐱")


mimi = Katze("Mimi", "schwarz")
mimi.essen()    # von Tier geerbt
mimi.miauen()  # eigene Methode
Mimi isst.
Mimi sagt: Miau! 🐱

Praktisches Beispiel: Bankkonto

pythonbankkonto.py
class Bankkonto:
    def __init__(self, inhaber, startguthaben=0):
        self.inhaber  = inhaber
        self._guthaben = startguthaben  # _ = privat (Konvention)
        self.transaktionen = []

    def einzahlen(self, betrag):
        if betrag > 0:
            self._guthaben += betrag
            self.transaktionen.append(f"+{betrag}€")
            print(f"✅ {betrag}€ eingezahlt. Guthaben: {self._guthaben}€")

    def abheben(self, betrag):
        if betrag > self._guthaben:
            print("❌ Nicht genug Guthaben!")
        else:
            self._guthaben -= betrag
            self.transaktionen.append(f"-{betrag}€")
            print(f"💸 {betrag}€ abgehoben. Guthaben: {self._guthaben}€")

    @property
    def guthaben(self):
        return self._guthaben


konto = Bankkonto("Anna", 1000)
konto.einzahlen(500)
konto.abheben(200)
konto.abheben(2000)
print(f"Transaktionen: {konto.transaktionen}")
✅ 500€ eingezahlt. Guthaben: 1500€
💸 200€ abgehoben. Guthaben: 1300€
❌ Nicht genug Guthaben!
Transaktionen: ['+500€', '-200€']
