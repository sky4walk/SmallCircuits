04

Warum das Skalarprodukt null wird

Bei φ = 90° ist cos(φ) = 0 — unabhängig davon, wie lang a und b sind. Zieh b so, dass es senkrecht auf a steht: die Orthogonalitäts-Anzeige oben schaltet auf orthogonal, sobald a · b praktisch null ist.

φ = 0° → cos(φ) = 1 → maximales Produkt, gleiche Richtung φ = 90° → cos(φ) = 0 → a · b = 0, a ⊥ b (orthogonal) φ = 180° → cos(φ) = -1 → negatives Produkt, entgegengesetzt

Genau dieses Kriterium — Skalarprodukt gleich null — entscheidet in der Quanteninformatik, ob zwei Zustände vollständig unterscheidbar sind (Abschnitt 05 im Artikel).

03

Einheitsvektor

Klick auf „Einheitsvektoren“: beide Pfeile werden auf Länge 1 skaliert, û = v / |v| — Richtung bleibt erhalten, nur die Länge ändert sich. So sehen Qubit-Zustände aus, die laut Artikel immer |α|² + |β|² = 1 erfüllen müssen.
