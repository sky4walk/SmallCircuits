⚗️ 5. Verbindungstypen

🔷 Alkane -an

Nur Einfachbindungen, CₙH₂ₙ₊₂
CH₃–CH₂–CH₂–CH₃ → Butan

🟢 Alkene -en

Eine C=C-Doppelbindung
CH₂=CH–CH₃ → Prop-1-en
CH₂=C(CH₃)–CH₃ → 2-Methylprop-1-en

🟣 Alkohole -ol

–OH-Gruppe, höchste Priorität
CH₃–CH(OH)–CH₃ → Propan-2-ol
CH₃–CH(OH)–CH(CH₃)–CH₃ → 3-Methylbutan-2-ol

🟠 Ketone -on

C=O in der Kette
CH₃–CO–CH₃ → Propan-2-on
CH₃–CO–CH(CH₃)–CH₃ → 3-Methylbutan-2-on

🔵 Carbonsäuren -säure

–COOH immer an Position 1 (keine Nummer nötig)
CH₃–COOH → Ethansäure
CH₃–CH(CH₃)–COOH → 2-Methylpropansäure

⚡ 1. Elektronegativität (EN)

Die Elektronegativität gibt an, wie stark ein Atom das Bindungselektronenpaar an sich zieht.

Entscheidend: ΔEN = |EN₁ − EN₂|

AtomENAtomEN
F4,0 (höchster Wert!)N3,0
O3,5C2,5
Cl3,2S2,5
Br2,7H2,2
🔑 Prüfungs-Merksatz: ΔEN → Bindungsart  |  Geometrie → Polarität  |  Teilchen → Wechselwirkungen

🔗 2. Bindungsarten nach ΔEN

Unpolare kovalente BindungΔEN = 0 – 0,4
Polare kovalente Bindung (Dipolbindung)ΔEN = 0,4 – 1,7
Ionenbindung (Salz)ΔEN > 1,7
ΔENBindungsartBeispiele
0 – 0,4Unpolare kovalente BindungH₂, O₂, CH₄, Cl₂
0,4 – 1,7Polare kovalente BindungH₂O, NH₃, HCl, CO₂
> 1,7Ionenbindung → SalzNaCl, KBr, CaO

🔵 3. Kovalente Bindung

Entsteht zwischen Nichtmetallen. Elektronen werden geteilt.

Unpolar kovalent  ΔEN ≤ 0,4

Elektronen gleichmäßig verteilt
Beispiele: H₂, O₂, Cl₂, CH₄

Polar kovalent  ΔEN 0,4–1,7

Elektronen zum elektronegativeren Atom verschoben → Partialladungen δ+ und δ−
Beispiele: H₂O, NH₃, HCl

🧂 4. Ionenbindung – Wann ist eine Verbindung ein Salz?

Metall + Nichtmetall, ΔEN > 1,7 → Elektronen werden übertragen, nicht geteilt.

Eigenschaften von Salzen

  • Bestehen aus Ionen (Kationen + Anionen)
  • Aufgebaut als Kristallgitter
  • Hohe Schmelztemperaturen
  • In Wasser löslich → elektrisch leitfähig

Beispiele

NaCl (ΔEN = 2,1) → Ionenbindung
KBr → Ionenbindung
CaO → Ionenbindung

Merke: Metall + Nichtmetall = fast immer Salz

🌊 7. Zwischenmolekulare Wechselwirkungen

Je stärker die Wechselwirkungen, desto höher der Siedepunkt.

① Wasserstoffbrücke – stark

H direkt an O, N oder F
+ freies EP am Nachbarmolekül

Beispiele: H₂O, NH₃, HF

② Dipol-Dipol – mittel

Zwischen polaren Molekülen
δ+ und δ− ziehen sich an

Beispiele: HCl, Aceton

③ London-Dispersion – schwach

Zwischen allen Teilchen
Kurzzeit-Dipole durch Elektronenbewegung
Stärke ↑ mit Molekülgröße

Beispiele: CH₄, Edelgase
TypVoraussetzungStärkeBeispiele
WasserstoffbrückeH an O, N oder F⬛⬛⬛ starkH₂O, NH₃, HF
Dipol-DipolPolare Moleküle⬛⬛ mittelHCl, Aceton
London-DispersionAlle Teilchen⬛ schwachCH₄, Edelgase
Warum ist Wasser flüssig, Methan gasförmig?
H₂O: H an O → Wasserstoffbrücken → starke Anziehung → flüssig
CH₄: unpolar → nur London-Dispersion → sehr schwach → gasförmig

Aufgabe 5 – Warum ist Wasser flüssig, Methan gasförmig?

Erkläre den Unterschied im Aggregatzustand bei Raumtemperatur.
Wasser: H an O → Wasserstoffbrücken → starke Anziehung zwischen Molekülen → bleibt flüssig
Methan: unpolar → nur London-Dispersion → sehr schwache Wechselwirkung → gasförmig

🔀 9. Wenn-Dann-Schema

① Bindung bestimmen

Wenn ΔEN > 1,7
→ Ionenbindung (Salz)
Wenn ΔEN 0,4 – 1,7
→ Polare kovalente Bindung (Dipolbindung)
Wenn ΔEN < 0,4
→ Unpolare kovalente Bindung

② Molekül polar?

Wenn polare Bindungen UND unsymmetrische Form
→ Molekül ist ein Dipol
Wenn symmetrisch (auch mit polaren Bindungen)
→ Kein Dipol

③ Zwischenmolekulare Kräfte

Wenn H an O, N oder F gebunden ist
→ Wasserstoffbrückenbindung (stärkste)
Wenn Molekül polar, aber kein H an O/N/F
→ Dipol-Dipol-Wechselwirkung
Wenn Molekül unpolar
→ London-Dispersion (schwächste)

📌 10. Spickzettel Elektronegativität

① EN-Werte

  • F = 4,0 (höchster!)
  • O = 3,5
  • N = 3,0
  • C/S = 2,5
  • H = 2,2

② ΔEN → Bindung

  • < 0,4 → unpolar
  • 0,4–1,7 → polar kovalent
  • > 1,7 → Ionenbindung

③ Dipol?

  • Polare Bindungen
  • UND unsymmetrisch
  • → dann Dipol

④ Wechselwirkungen

  • H an O/N/F → H-Brücke
  • Dipol → Dipol-Dipol
  • Unpolar → London

⑤ Kein Dipol (trotz polarer Bind.)

  • CO₂ (linear)
  • CCl₄, CH₄ (tetraedrisch)
  • BF₃ (trigonal-planar)

⑥ Merksatz

ΔEN → Bindung
Geometrie → Polarität
Teilchen → Wechsel­wirkungen
